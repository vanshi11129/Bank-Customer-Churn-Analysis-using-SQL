# Bank-Customer-Churn-Analysis-using-SQL

# Poverty Analysis in India

An data analytics project analyzing a customer churn dataset of a bank . This project combines Python for data cleaning and Machine Learning using XGBoost along with SQL for a deeper analysis of the churn rate and what co-relations it could have with other factors.


**🚀 Project Overview
**


Customer churn is the rate at which customers stop doing business with a company or cancel their subscriptions over a specific period of time. Knowing what rate customers leave your business is important so as to make new policies to retain the customers which are at risk of leaving.  

**Key Objectives:
**

1.Clean and restructure government data for analysis.

2.Create a model for prediction using XGBoost. 

3. Perform further analysis using SQL and SSMS.

**🛠️ Tech Stack 
**

Data Source: Dataset by Kaggle

Data Cleaning: Python (Pandas).

Machine Learning: Python (XGBoost)

Database and Quering: SQL


**⚙️ Data Pipeline & Methodology
**

**1. Data Cleaning (Python)
**


A. Dropped irrelevant features that do not contribute to predictive power (e.g. CustomerId).

B. Encoded categorical variables (Geography and Gender) using One-Hot Encoding and by Creating dummies. 

**2. Creating of a model for prediction 
**


A. Splitting the Dataset into X and y 

B. Giving hyperparameters to balance out the imbalanced dataset 

C. Testing the model

**3. Analysis using SQL
**


A. Deep analysis by asking questions such as : 
	What is the total churn rate? 
	Correlation between inactive members and churn
	Churn rate per age and gender etc
